## Aprendizaje por Refuerzo para la Detección de Anomalías mediante una Aplicación Web

### Descripción de la tarea
Se deberá desarrollar una aplicación web que funcione como un recurso educativo sobre Aprendizaje por Refuerzo. La aplicación debe estar orientada a una audiencia técnica o semi-técnica y debe explicar el tema utilizando texto, imágenes, diagramas, animaciones, simulaciones o cualquier otro recurso creativo

### Contenido mínimo requerido

- Introducción al Aprendizaje por Refuerzo.
- Descripción de los conceptos fundamentales (agente, entorno, estado, acción, recompensa y política).
- Explicación del proceso de aprendizaje mediante recompensas.
- Descripción de al menos un algoritmo o enfoque de RL.
- Caso práctico de detección de anomalías.
- Visualización o simulación del comportamiento del agente.
- Conclusiones y limitaciones de la propuesta.

### Requisitos técnicos
- Aplicación web ejecutable en navegador.
- Interfaz gráfica clara y navegable.
- Diseño responsive o adaptable a distintas resoluciones.
- Uso de HTML/CSS/JavaScript u otros frameworks web.
- Código fuente almacenado en GitHub.
- README con instrucciones de instalación y ejecución.
- Uso adecuado de referencias bibliográficas.
